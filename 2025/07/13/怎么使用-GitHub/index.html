怎么使用 GitHub

什么是 GitHub

如果你只是想注册，那么你可以跳过这部分。

了解 GitHub 之前，我们先了解一下 Git。

那 Git 从哪克隆东西呢，总不能所有东西都在不同的地方吧。所以要有个中心（Hub），所以 GitHub 就诞生了。

当然了，GitLab，Gitcode，也是一种 Git 的中心，只不过最出名的还是 GitHub。

怎么在国内访问 GitHub 官网

  1. 有的优质运营商直接就能访问 GitHub。
  2. 科学上网。
  3. 下载 Watt Toolkit(原名 Steam++)，打开，点侧边栏的闪电（网络加速），记得勾选 GitHub 加速，点击”一键加速“。

怎么注册 GitHub 账号

  1. 打开 GitHub。
  2. 点击右上角的“Sign up”。
  3. 输入个人信息（才你想要：有道翻译。
  4. 打开邮箱，点击 GitHub 发送的邮件，复制验证码，输入验证码。
  5. 如果它问你问题，认真回答。
  6. 注册完成后，您可以进一步设置您的账户信息。包括上传头像、填写个人简介、选择关注的开发者或项目等。这些信息将展示在您的GitHub主页上，帮助其他用户更好地了解您。

仓库怎么用

仓库是什么

仓库是用来存储代码的地方。

仓库怎么用

仓库有两种类型：公开仓库和私有仓库。
公开仓库：任何人都可以访问。
私有仓库：只有你自己可以访问。

  1. 打开 GitHub，登录你的账号。
  2. 点击右上角的“+”号，选择“New repository”。
  3. 填写仓库名称，仓库描述，选择仓库类型，选择仓库许可证。其实只用填写仓库名称就可以了，用不到的默认就可以了。
  4. 点击“Create repository”。
  5. 下载GitHub Desktop，或者配置Git。
  6. 打开 GitHub Desktop，点击“File”，选择“Clone repository”。
  7. 选择你要克隆的仓库，点击“Clone”。
  8. 克隆完成后，你可以在本地修改代码，然后推送到 GitHub 上。
  9. 推送到 GitHub 上，点击“Push origin”。
  10. 推送完成后，你可以在 GitHub 上查看你的代码。
  11. 如果有问题，你可以在 GitHub 上提 Issue。
  12. 如果你想修改别人仓库的代码，你可以在 GitHub 上提 Pull Request。
  13. 建议你为你的仓库添加 README.md 文件，描述你的仓库是什么，怎么用，怎么安装，怎么配置。

完结撒花！
